1. жҖ»дҪ“жЎҶжһ¶

1.1. з®Җд»Ӣ

жң¬ж–ҮжЎЈжұҮжҖ»е№¶жҖ»з»“жҲ‘жүҖдҪҝз”Ёзҡ„emacsй…ҚзҪ®зҡ„еҹәжң¬з”Ёжі•,е№¶дјҡиҝӣиЎҢдёҚй—ҙж–ӯең°жӣҙж–°гҖӮ

з”ұдәҺжҲ‘еёҢжңӣеңЁеӨҡдёӘе№іеҸ°дёҠдҪҝз”Ёд»–пјҢе…¶дёӯпјҢеҸҲеӣ еҸ—йҷҗдәҺи®әж–ҮеҶҷдҪңгҖҒз”»еӣҫгҖҒQQгҖҒеҫ®дҝЎзӯүе·ҘдҪңжөҒзҡ„йңҖжұӮпјҢжң¬emacsй…ҚзҪ®дёҚеҫ—дёҚж”ҜжҢҒwindowsзі»з»ҹдёҠзҡ„дҪҝз”ЁгҖӮ

з”ұдәҺеңЁжңҚеҠЎеҷЁз«ҜеҶҷд»Јз Ғзҡ„йңҖжұӮпјҢйңҖиҰҒз»ҙжҢҒеңЁlinuxз»Ҳз«ҜдёҠзҡ„дҪҝз”ЁгҖӮз”ұдәҺеңЁз¬”и®°жң¬з”өи„‘дёҠдҪҝз”Ёlinuxзҡ„йңҖжұӮпјҢиҝҳйңҖиҰҒз»ҙжҢҒlinuxжЎҢйқўз«Ҝзҡ„дҪҝз”ЁгҖӮеӣ иҖҢжҖ»дҪ“дёҠйңҖиҰҒд»ҘдёӢеҮ йғЁеҲҶпјҡ

  1. еңЁwindowsзі»з»ҹдёҠпјҢдё»иҰҒеҠҹиғҪйңҖжұӮеҢ…жӢ¬е’ҢдёӨйғЁеҲҶ;
  2. еңЁlinuxжңҚеҠЎеҷЁдёҠпјҢе…·жңүиүҜеҘҪзҡ„pythonиҜӯжі•ж”ҜжҢҒпјӣ
  3. еңЁwindowsзі»з»ҹе’Ңlinuxзі»з»ҹдёӢпјҢжңҖеҘҪжӢҘжңүд»ҘдёӢж”ҜжҢҒпјҡ
    1. йӮ®д»¶еӨ„зҗҶ
    2. C++гҖҒCиҜӯжі•ж”ҜжҢҒпјӣ
    3. еҗ„з§Қзј–зЁӢиҜӯиЁҖзҡ„ејҖз®ұеҚіз”Ёпјӣ

й’ҲеҜ№дәҺд»ҘдёҠй—®йўҳпјҢзӣ®еүҚжүҖйҮҮз”Ёзҡ„жҠҳдёӯи§ЈеҶіж–№жЎҲжҳҜпјҡйҮҮз”Ёwin10зі»з»ҹ+ WSL2пјҢжүҖжңүзҡ„emacsй…ҚзҪ®еқҮе»әз«ӢеңЁlinuxзі»з»ҹдёҠиҝӣиЎҢй…ҚзҪ®пјҢеҗҢж—¶дёәwsl2ж’°еҶҷдёҖдәӣеҮҪж•°пјҢд»ҘжҸҗдҫӣжӣҙеҠ ж–№дҫҝзҡ„зі»з»ҹдәӨдә’гҖӮ йҖҡиҝҮд»ҘдёҠж–№ејҸпјҢжҲ‘ж•ҙеҗҲдәҶжҲ‘зҡ„е·ҘдҪңжөҒпјҢе®һзҺ°дёҠиҝ°йңҖжұӮпјҡ

  1. йҖҡиҝҮеңЁwsl2дёҠпјҲзӣ®еүҚйҮҮз”Ёubuntu20.04пјүйҖҡиҝҮи„ҡжң¬е®үиЈ…дәҢиҝӣеҲ¶иҪҜд»¶пјҢе°ҶemacsдҪңдёәеүҚз«ҜпјҢе®һзҺ°emacsдёӯзҡ„еҗ„йЎ№еҹәжң¬еҠҹиғҪпјӣ
  2. йҖҡиҝҮtrampпјҢдёҖдёӘsshиҪҜд»¶пјҢжқҘиҮӘеҠЁз®ЎзҗҶиҝһжҺҘиҝңзЁӢжңҚеҠЎеҷЁпјӣ
  3. йҖҡиҝҮжү§иЎҢpowershellе‘Ҫд»ӨпјҢжқҘжӣҙеҘҪең°иҫ…еҠ©wslе’Ңwindowsзі»з»ҹд№Ӣй—ҙзҡ„дәӨжөҒпјӣ

еҗҢж—¶пјҢжҲ‘иҝҳдёәemacsзҡ„дҪҝз”ЁеҮҶеӨҮдәҶдёӨз§ҚзҺҜеўғпјҡ

  1. TUIпјҲText User InferfaceпјүпјҢйҖҡиҝҮеңЁwindowsзҡ„е®ҳж–№еә”з”ЁеёӮеңәдёӢиҪҪterminalпјҢеҸҜд»ҘзӣҙжҺҘе»әз«ӢеҜ№wslзҡ„и®ҝй—®гҖӮжҲ‘еёёеёёеңЁterminalдёӯеҗҜеҠЁemacsпјҢиҝҷеҚ жҚ®дәҶжҲ‘е·ҘдҪңдёӯзҡ„з»қеӨ§еӨҡж•°ж—¶й—ҙгҖӮжҜ”иө·GUIзҺҜеўғпјҢterminalжңүд»ҘдёӢдјҳзӮ№пјҡ
    1. йҖҹеәҰжӣҙеҝ«пјҢжҲ‘зҡ„emacsеҫҲйңҖиҰҒй«ҳж•ҲзҺҮпјҢжӯЈеёёз”ЁзқҖз”ЁзқҖпјҢе°ұдјҡејҖеҫҲеӨҡbufferпјҢеҰӮжһңиҝҳжү“ејҖlspиҝҷз§ҚжңҚеҠЎпјҢйӮЈд№Ҳе°ұжӣҙеӨҚжқӮдәҶпјӣ
    2. tuiзңӢиө·жқҘжӣҙеҘҪзңӢпјӣ
    3. tuiжӣҙеҠ зәҜзІ№пјҢеӣ дёәз»қеӨ§еӨҡж•°ж—¶еҖҷпјҢжҲ‘йғҪжҳҜеңЁcodingпјӣ
  2. GUIзҺҜеўғпјҢwin11еҸҜд»Ҙж”ҜжҢҒWSLдёӢзҡ„еҺҹз”ҹGUIпјҢдёҚиҝҮеҫҲеҸҜжғңпјҢжҲ‘е·ҘдҪҚзҡ„з”өи„‘еңЁзЎ¬д»¶дёҠж— жі•ж”ҜжҢҒwin11пјҢжүҖд»ҘжҲ‘еҸӘиғҪдҪҝз”Ёwin10пјҢиҖҢеңЁwin10дёӯпјҢжҲ‘йҮҮз”ЁmobaxtermпјҢдёҖдёӘејҖжәҗзҡ„sshиҝһжҺҘиҪҜд»¶пјҢжқҘиҝӣиЎҢGUIжҳ е°„гҖӮ mobaxtermиҮӘеёҰx11пјҢеңЁз»Ҳз«Ҝй”®е…ҘemacsпјҢе°ұеҸҜд»Ҙдә§з”ҹдёҖдёӘguiзүҲжң¬зҡ„emacsдәҶгҖӮдҪҝз”ЁGUIзҡ„ж–№ејҸжңүд»ҘдёӢдјҳзӮ№пјҡ
    1. ж”ҜжҢҒеӣҫзүҮпјҢеҰӮжӯӨеңЁеҶҷеҚҡе®ўж—¶пјҢеҸҜд»ҘеҶ…зҪ®зҺ°е®һеӣҫзүҮпјӣеңЁеҶҷlatexж—¶пјҢеҸҜд»ҘиҜ•иҜ•жҹҘзңӢдә§з”ҹзҡ„pdfпјҲtuiдёҚиЎҢпјүпјӣеңЁpythonз§‘з ”з»ҳеӣҫж—¶пјҢд№ҹеҸҜд»ҘзӣҙжҺҘdisplayеҫ—еҲ°еҜ№еә”зҡ„з»“жһңпјӣ
    2. йј ж ҮдҪҝз”ЁгҖӮguiж”ҜжҢҒдҪҝз”Ёйј ж ҮпјҢиҝҷеңЁжӢјеҶҷжЈҖжҹҘгҖҒз»ҳеӣҫзӯүдәӢжғ…дёҠжңүдёҖе®ҡз”ЁйҖ”пјӣ
    3. жӣҙеӨҡзҡ„themeгҖӮжңүдёҖдәӣthemeиҝҳжҳҜguiжҜ”иҫғеҘҪзңӢ
    4. еӯ—дҪ“пјҢз•ҘиҝҮдёҚи°Ҳпјӣ
    5. жІЎжңүеҝ«жҚ·й”®зјәжҚҹзҡ„й—®йўҳгҖӮеңЁtuiдҪҝз”Ёж—¶пјҢдёәemacsе®ҡд№үзҡ„еҝ«жҚ·й”®пјҢеҸҜиғҪдјҡдёҺз»Ҳз«Ҝзҡ„еҝ«жҚ·й”®еҸ‘з”ҹеҶІзӘҒгҖӮиҖҢжҲ‘иҮід»ҠжІЎжүҫеҲ°жҖҺд№ҲжҠҠterminalдёӯжүҖжңүзҡ„еҝ«жҚ·й”®йғҪе…іжҺүвҖҰвҖҰ

2.3.1. insert codes

зӣёеҪ“дәҺеҶҷдәҶдёҖзӮ№зӮ№зҡ„иҜӯжі•зі–пјҢжҲ‘жҠҠе®ғз»‘е®ҡеңЁдәҶвҖңC-c s iвҖқдёҠпјҢд»ЈиЎЁ srcпјҢinputгҖӮ

;; 设置org快速插入源代码
(defun org-insert-src-block (src-code-type)
        "Insert a `SRC-CODE-TYPE' type source code block in org-mode."
        (interactive
         (let ((src-code-types
                '("emacs-lisp" "python" "C" "sh" "java" "js" "clojure" "C++" "css"
                  "calc" "asymptote" "dot" "gnuplot" "ledger" "lilypond" "mscgen"
                  "octave" "oz" "plantuml" "R" "sass" "screen" "sql" "awk" "ditaa"
                  "haskell" "latex" "lisp" "matlab" "ocaml" "org" "perl" "ruby"
                  "scheme" "sqlite")))
           (list (ido-completing-read "Source code type: " src-code-types))))
        (progn
          (newline-and-indent)
          (insert (format "#+BEGIN_SRC %s\n" src-code-type))
          (newline-and-indent)
          (insert "#+END_SRC\n")
          (previous-line 2)
          (org-edit-src-code)))

;; (message "--------------------begin--------------------")
;;将其加载在快捷键上
(add-hook 'org-mode-hook
                '(lambda ()
                   ;; ;; C-TAB for expanding
                   ;; (local-set-key (kbd "C-<tab>")
                   ;; 'yas/expand-from-trigger-key)
                   ;; keybinding for editing source code blocks
                   (local-set-key (kbd "C-c s e")
                   'org-edit-src-code)
                   ;; keybinding for inserting code blocks
                   (local-set-key (kbd "C-c s i") 'org-insert-src-block )))

еҪ“жҲ‘еҶҷи®әж–Үж—¶пјҢиҝҷдёӘеҠҹиғҪеҫҲйҮҚиҰҒгҖӮ

жҲ‘дё»иҰҒдҪҝз”ЁдәҶдёӨз§Қcheckе·Ҙе…·пјҢдёҖдёӘжҳҜispellпјҢдёҖдёӘжҳҜflyspellгҖӮжҲ‘дёҚжҳҜзү№еҲ«дәҶи§Јиҝҷе„ҝиҝҷзҡ„еҢәеҲ«е’ҢиҒ”зі»пјҢдҪҶжҳҜж„ҹи§үflyspellжӣҙеҠ еӨҚжқӮдёҖзӮ№пјҢеҗҢж—¶жҲ‘е–ңж¬ўзҡ„жҳҜз®ҖеҚ•зҡ„дёңиҘҝгҖӮйҰ–е…Ҳд»Ӣз»ҚispellгҖӮispellжҳҫз„¶пјҢд№ҹжҳҜlinuxдёӢжҹҗжӢјеҶҷжЈҖжҹҘе·Ҙе…·aspellзҡ„emacsеүҚз«ҜпјҢзӣҙжҺҘM-xиҝҗиЎҢеҚіеҸҜгҖӮ

е’ҢеӨ§еӨҡж•°GNUе·Ҙе…·дёҖж ·пјҢе…¶дҪҝз”ЁеҫҲз®ҖеҚ•пјҢеҪ“дҪ жү“ејҖд№ӢеҗҺпјҢе®ғдҫҝдјҡдёҖдёӘдёӘзҡ„и®©дҪ е®Ўйҳ…йӮЈдәӣеҚ•иҜҚпјҢе№¶дёәжҜҸдёҖдёӘеҚ•иҜҚжҸҗдҫӣеҸҜжӣҝжҚўзҡ„йҖүйЎ№гҖӮйҖҡиҝҮиҝҷз§Қж–№ејҸпјҢе°ұеҸҜд»Ҙе®һзҺ°е…Ёж–Үзҡ„дёҖдёӘжӢјеҶҷжЈҖжҹҘдәҶгҖӮ

еҸҰдёҖдёӘе·Ҙе…·жҳҜflyspellпјҢжҚ®иҜҙеҠҹиғҪжӣҙеҠ ејәеӨ§пјҢеҪ“дҪ жү“ејҖиҝҷдёӘmodeд№ӢеҗҺпјҢд»–дјҡе°ҶжүҖжңүжңүй—®йўҳзҡ„wordдёӢж ҮеҸ‘зәўгҖӮжҲ‘дёҖзӣҙжІЎеӯҰжҳҺзҷҪжҖҺд№ҲиҝӣиЎҢжңҖеҹәжң¬зҡ„correctпјҢжүҖд»ҘиҝҷдёӘе·Ҙе…·жёёжҲҸең°ж–№ дҪҝз”ЁжҲ‘е°ұдёҚеҫ—иҖҢзҹҘдәҶгҖӮ

2.8. иҫ“е…Ҙжі•

emacsд№ҹиҮӘеёҰдәҶдёҖдәӣиҫ“е…Ҙжі•пјҢжқҘеә”еҜ№дёҖдәӣжһҒз«Ҝжғ…еҶөгҖӮжҜ”еҰӮжҲ‘зҺ°еңЁдҪҝз”Ёзҡ„wslпјҢеҰӮжһңжҲ‘йҖҡиҝҮX11жү“ејҖпјҢйӮЈд№Ҳе°ұйңҖиҰҒиҝҷдёӘеӯҗзі»з»ҹй…ҚзҪ®иҫ“е…Ҙжі•пјҢеҗҰеҲҷе°ұж— жі•иҫ“е…Ҙдёӯж–ҮпјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢwindowsзі»з»ҹй»ҳи®Өзҡ„иҫ“е…Ҙжі•жҳҜж— жі•дҪҝз”Ёзҡ„гҖӮ

жҲ‘дёәжҲ‘зҡ„emacsй…ҚзҪ®жҸҗдҫӣдәҶдёӨз§Қиҫ“е…Ҙжі•пјҢдёҖз§ҚжҳҜpyimпјҢиҝҷз§Қиҫ“е…Ҙжі•жҳҜelispе®һзҺ°зҡ„пјҢжүҖд»ҘдёҚйңҖиҰҒйўқеӨ–е®үиЈ…д»»дҪ•иҪҜд»¶пјҢеҸҜйқ жҖ§еҫҲејәгҖӮдҪҶжҳҜзЎ®е®ҡе°ұжҳҜпјҢиҫ“е…Ҙжі•дёҚеӨӘеҘҪз”ЁвҖҰвҖҰеҸҰдёҖз§Қж–№ејҸжҳҜпјҢе®үиЈ…дёҚйңҖиҰҒжҠҳи…ҫзҡ„иҫ“е…Ҙжі•еҗҺз«ҜдәҺlinuxдёӯпјҢе№¶еңЁemacsдёӯжҸҗдҫӣеҜ№еә”зҡ„еүҚз«ҜгҖӮжҲ‘зӣ®еүҚдё»иҰҒдҪҝз”ЁиҝҷдёҖз§Қж–№жЎҲпјҢд№ҹе°ұжҳҜзӢ—е“ҘжүҖз»ҷеҮәзҡ„rimeеүҚз«ҜгҖӮжҲ‘зҡ„й…ҚзҪ®еҸӮиҖғзҡ„жҳҜlazycatзҡ„еҚҡе®ўпјҢеҰӮдёӢпјҡ
