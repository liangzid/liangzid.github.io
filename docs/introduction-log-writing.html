introduction怎么写?

introduction 的撰写可简略地分为以下步骤：

  1. 第一段，背景。展示当前工作的重要性；
  2. 第二段，阐述当前背景下的一个具体的问题；
  3. 第三段，介绍一下已有的工作；
  4. 第四段，阐述已有工作的缺点；
  5. 第五段，介绍我们的工作的基本思路和优势；
  6. 第六段，总结已有贡献，最好有数值等形式的具体结果。
